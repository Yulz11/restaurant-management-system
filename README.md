# 🍽️ Restaurant Management System - Backend Engine

A robust, containerized RESTful API designed to manage restaurant menus, built with high-performance modern technologies. This project demonstrates full-stack architectural principles, from database orchestration to API design.

## 🚀 Key Features
* **Automated Data Persistence:** Integrated with MySQL for reliable storage.
* **Containerized Infrastructure:** Orchestrated with Docker & Docker Compose for seamless environment consistency.
* **Interactive Documentation:** Built-in Swagger UI for real-time API testing and exploration.
* **Asynchronous Performance:** Leverages FastAPI for high-speed request handling.

## 🛠️ Tech Stack
* **Language:** Python 3.12
* **Framework:** FastAPI
* **ORM:** SQLAlchemy
* **Database:** MySQL 8.0
* **DevOps:** Docker, Docker Compose

## 📦 Quick Start
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Yulz11/restaurant-management-system.git](https://github.com/Yulz11/restaurant-management-system.git)
    ```
2.  **Spin up the environment:**
    ```bash
    docker-compose up -d
    ```
3.  **Access the API:**
    Open `http://localhost:8000/docs` to explore the interactive Swagger documentation.
